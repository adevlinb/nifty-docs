Dog Detail API - GET, DELETE - /api/dogs/id/

Use this endpoint to:
  • retrieve a dog with info and two photos
  • compare altered and unaltered photos
  • delete the dog

GET /api/dogs/id/

This endpoint will:
  • retrieve a dog with info and two photos
  • compare altered and unaltered photos

GET http://localhost:8000/api/dogs/4/
